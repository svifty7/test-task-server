Тестовое задание

Необходимо использовать Vue.js и JavaScript (без jQuery)

Первым делом, необходимо сделать GET запрос сюда: 
http://elma-test-task.tmweb.ru/radial-menu
и получить json с названиями пунктов меню (можно использовать библиотеки, например, axios)

GET-параметры

  • ?length=<number>
    - вместо <number> используйте свое значение (от 4 до 10), если нужно какое-то конкретное количество элементов в меню

Следующим шагом нужно построить динамическое радиальное меню на основе получаемого json массива:

  • Меню должно перетаскиваться по странице от края до края (нативным методом)
  • Если часть меню скрывается за пределами экрана, то при наведении на меню необходимо иметь возможность колесиком прокрутить и увидеть необходимый пункт меню
  • Содержимое пункта меню (текст и иконка) всегда должно оставаться в обычной ориентации, т.е. текст должен легко читаться слева-направо
  • Иконки должны быть в svg с возможностью изменения цвета в сочетании с текстом (иконки(у) можно взять любые на свой вкус или с макета)
  • Стили лучше всего писать на Scss или Less - на свое усмотрение. Мы используем первое :)

Формулы для построения

  • Координаты точки на окружности getDegreePosition(angle, radius) { return { x: Math.cos(degToRad(angle)) * radius, y: Math.sin(degToRad(angle)) * radius } }
  • Конвертирование degrees в radians degToRad(angle) { return angle * (Math.PI / 180) }
